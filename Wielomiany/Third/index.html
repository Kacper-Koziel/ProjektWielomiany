Twierdzenia wielomianów 3

Zaczynajmy

f(x) = 2x4 - 3x3 - 11x2 + 12x + 18

Krok 1: Twierdzenie o reszcie

Na podstawie twierdzenia o reszcie, możliwe pierwiastki wielomianu znajdują się wśród dzielników wyrazu wolnego (18)
podzielonych przez dzielniki współczynnika przy najwyższej potędze (2).

Dzielniki 18: ±1, ±2, ±3, ±6, ±9, ±18.

Dzielniki 2: ±1, ±2.

Dalej >>

Potencjalne pierwiastki to: ±1, ±1/2, ±2, ±3, ±3/2, ±6, ±9, ±18

Krok 2: Sprawdzenie pierwiastków

Dla x = 3:

f(3)= 2(3)4 - 3(3)3 - 11(3)2 + 12(3) + 18

f(3) = 2(81) - 3(27) - 11(9) + 36 + 18 = 162 - 81 - 99 + 36 + 18 = 36 - 36 = 0

Zatem x = 3 jest miejscem zerowym.

Dalej >>

Krok 3: Dzielenie przez x - 3

Skoro x = 3 jest miejscem zerowym, to dzielimy f(x) przez x - 3.
Użyjemy dzielenia syntetycznego:

2x4 - 3x3 - 11x2 + 12x + 18 : (x - 3)

Wynik dzielenia:
f(x) = (x - 3)(2x3 + 3x2 - 2x - 6)

Dalej >>

Krok 4: Rozkład wielomianu sześciennego

Q(x) = (2x3 + 3x2 - 2x - 6)

Możliwe pierwiastki: ±1, ±2, ±3, ±6, ±1/2, ±3/2

Q(-2) = 2(-8) + 3(4) - 2(-2) - 6 = -16 + 12 + 4 - 6 = 0

Zatem x = -2 jest pierwiastkiem.

Dalej >>

Krok 5: Dzielenie przez x + 2 oraz delta

Dzielimy Q(x) przez x + 2:

Q(x)= (x + 2)(2x2 - x - 3)

Rozkład części kwadratowej 2x2 - x - 3

Δ = b2 - 4ac = (-1)2 - 4(2)(-3) = 1 + 24 = 25
√Δ = 5

Dalej >>

Krok 6: Obliczenie miejsc zerowych i podsumowanie wyników

Dla +√Δ: x = 3/2

Dla -√Δ: x = -1

Ostateczny rozkład tego wielomianu:

f(x)=(x - 3)(x + 2)(2x - 3)(x + 1)

Miejsca zerowe: x1 = 3, x2 = -2, x3 = 3/2, x4 = -1
