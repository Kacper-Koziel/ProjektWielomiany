Twierdzenia wielomianów

Zaczynajmy

P(x) = x3 - 6x2 + 11x - 6

Krok 1: Próbujemy znaleźć miejsca zerowe za pomocą twierdzenia o reszcie

Jeśli P(x) ma miejsce zerowe w x = a, to reszta z dzielenia P(x) przez a wynosi zero,
czyli: P(a) = 0

P(x) = x3 - 6x2 + 11x - 6 => a = 6

Dzielniki a: 1, 2, 3, -1, -2, -3, -6

Dalej >>

Dzielniki a: 1, 2, 3, -1, -2, -3, -6

Krok 2: Sprawdzamy, czy dla którejkolwiek z tych wartości P(x) = 0

P(-1) = -1 - 6 - 11 - 6 ≠ 0

P(-2) = -8 - 24 - 22 - 6 ≠ 0

P(1) = 1 - 6 + 11 - 6 = 0 => Mamy nasze pierwsze miejsce zerowe!

x1 = 1

Dalej >>

Krok 3: Podzielmy wielomian przez (x - 1)

Teraz możemy podzielić wielomian przez x-1 za pomocą dzielenia wielomianów.

P(x) : (x - 1) = (x2 - 5x + 6)

P(x) = (x - 1)(x2 - 5x + 6)

Dalej >>

Krok 4: Rozkładamy kwadratowy wielomian

Teraz mamy do czynienia z kwadratowym wielomianem (x2 - 5x + 6)
Możemy go rozłożyć na czynniki:

(x2 - 5x + 6) = (x - 2)(x - 3)

~Delta na następnej stronie

Dalej >>

Krok 4.1 (5): Delta

delta = (-5)2 - 4(6 x 1)

delta = 25 - 24 = 1

x2 = (5 + 1) / 2 = 2

x3 = (5 - 1) / 2 = 3

Dalej >>

Krok 6: Podsumowanie wyników

Ostateczny rozkład wielomianu to:

P(x) = (x - 1)(x - 2)(x - 3)

Zatem miejsca zerowe tego wielomianu to:

x1 = 1, x2 = 2
x3 = 3
